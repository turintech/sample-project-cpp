# sample-project-cpp
This is a sample project for showcasing how optimization can be performed in Python using Artemis

## Usage

Build:

```shell
make && make compile
```

Run Main:

```shell
./build/src/main
```

Run Unit Tests:

```shell
./build/tests/tests [test]
```

Run Benchmarking:

```shell
./build/tests/tests [benchmark]
```

For more advanced settings, run cmake directly.
